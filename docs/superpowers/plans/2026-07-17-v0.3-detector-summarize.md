# mandor v0.3 — Detector + Summarize + Spool Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: superpowers:executing-plans. Checkbox steps.

**Goal:** Turn worker deaths and resource anomalies into deduplicated incident bundles (stable JSON schema v1) written to `<state-dir>/incidents/`, with Go/Rust/Python trace extraction and heuristic verdicts.

**Architecture:** All detection is pure logic over data the supervisor already holds (exit status, restart history, stats window, log ring). On death: classify cause → extract trace from ring tail → signature-dedup → serialize bundle → tmp+rename into the spool dir. On sampler tick: leak heuristic over the RSS window emits `leak-suspect` incidents (rate-limited). Wall-clock ISO-8601 comes from CLOCK_REALTIME + pure civil-date math (no libc).

**Global constraints:** as before. Bundle schema below is a STABLE CONTRACT (premium sidecar depends on it); any change bumps `"v"`.

## Bundle schema v1 (locked)

```json
{"v":1,"ts":"2026-07-17T22:47:03Z",
 "process":{"name":"api","cmd":"./api --port 8080","pid":42,"restarts":3},
 "cause":"signal:SIGSEGV" | "exit:1" | "restart-loop" | "leak-suspect",
 "trace":{"lang":"go|rust|python|unknown","frames":["pkg.fn file.go:123"],"raw":"..."},
 "logs_tail":["[E!] panic: nil deref","[O] listening on :8080"],
 "stats_timeline":[{"t":"-60s","rss_mb":812,"cpu_pct":97}],
 "verdict":"RSS grew 40MB/min for 12min before exit"}
```

`logs_tail` entry prefix: `[O]`/`[E]` stream marker, `!` appended when the line
looks error-ish (`panic|error|fatal|exception|traceback|warn`, case-insensitive).
Last ≤200 lines. `trace.raw` ≤4 KB, `frames` ≤16.

## Heuristics (locked constants, tunable later via config in v0.4)

- restart-loop: ≥5 restarts of a worker within 5 min → cause `restart-loop`
  (fires when crossed; re-arms after 10 min without a death).
- leak-suspect: RSS strictly grew in ≥6 consecutive samples AND net growth
  ≥32 MB over the window → `leak-suspect`, at most once per 30 min per worker.
- Signature dedup: signature = FNV-1a of (cause kind, worker name, first
  error-ish line with digits/hex stripped). Identical signature within 10 min
  → suppressed (in-memory count bump only).

- `src/summarize.zig` — errorish(), signature hash, trace dispatch, verdict
  builder (pure, tested).
- `src/detector.zig` — restart-loop window, leak window walk, dedup/rate-limit
  state (pure, tested).
- `src/spool.zig` — ISO-8601 (pure civil-date math, tested), bundle
  serializer (pure, golden-tested), incident file writer (Linux).
- `src/supervisor.zig` — wire death path + tick path.
- Fixtures in `test/fixtures/` from real Go/Rust/Python crash output; harness
  case asserting a bundle appears with correct cause.
