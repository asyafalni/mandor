Notes: `frames` stay strings (structured frames = v0.6, will bump to v3).
`cause_str` mirrors v1's cause for sidecar transition. `kind` ∈ exit | signal
| oom | restart-loop | leak-suspect. Env redaction: value replaced with
`<redacted>` when the NAME contains SECRET/TOKEN/PASSWORD/KEY/CREDENTIAL
(case-insensitive); env capped to 32 vars / 2 KB.

## Decisions

- Ring record grows a wall-clock ms timestamp: header [u16 len][u8 flags][u64 t_ms].
- `core_dumped` = wait-status bit 0x80, stored on Worker.
- `exe` resolved by the parent at spawn (same PATH walk as the child, via
  faccessat X_OK); `cwd` = mandor's own cwd (workers inherit it); ulimit
  nofile via getrlimit (inherited); memory_max from /sys/fs/cgroup/memory.max.
- `spawned_at` wall epoch stored at spawn alongside monotonic last_start_ms.
- `build.release` = MANDOR_RELEASE env, else GIT_SHA env, else omitted.
- `--stop-grace=DUR` (default 10s): after TERM/INT forward, SIGKILL leftovers
  at deadline. Second signal still escalates immediately.
- `--expected-exit=CODES` (e.g. "143,0"): listed codes behave exactly like
  exit 0 everywhere (restart policy, incidents, worst-code propagation).
  TOML keys: `stop_grace`, `expected_exit` (both strings).
- Incident emitters receive the whole worker table for `siblings`.
